### HW1 - 二手车价格预测
- **任务**：使用深度学习模型预测二手车价格
- **数据集**：used_car_train.csv, used_car_test.csv
- **状态**：✅ 已完成
- **模型性能**：验证集 R² = 0.9832，MAE ≈ 1,250

- ✅ 九维度训练监控：损失曲线、学习率调整、梯度变化等
- ✅ 残差分析显示模型预测合理，无明显偏差
- ✅ 总训练时间约100秒，高效稳定

- ✅ 学习曲线分析：验证模型泛化能力
